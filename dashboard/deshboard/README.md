# 🖥️ Módulo de Visualización (Dashboard)

Este directorio contiene el código fuente de la interfaz web interactiva del proyecto **Maternidad Informada Puno**.

## 🛠️ Tecnologías utilizadas
- **Streamlit**: Framework para la aplicación web interactiva.
- **Folium / Streamlit-Folium**: Para la renderización del mapa geoespacial de Puno.
- **Pandas / NumPy**: Para el procesamiento de los datos del IVI.

## 🚀 Cómo ejecutar el prototipo localmente

1. Asegúrese de tener Python 3.8+ instalado.
2. Instale las dependencias desde la raíz del proyecto:
   ```bash
   pip install -r requirements.txt
